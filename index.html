How to dectect adblockers with Js

Note: You might want to try multiple solutions.

1. Solution One

This solution relys on the fact that most adblockers will routinely block access to a file with ads in their name.

  1. Step One:
    Create a file named ads.js in the same folder as the html page that you want to detect and put the content below into it.
// ads.js
var adBlockerDectected = false;
  1. Step Two
    Then in your index.html or any other html page put the content below into it.
<!DOCTYPE HTML>
<html>
	<head>
		<script>
			var adBlockerDectected = true;
		</script>
		<!-- Link to this file to set adBlockerDectected to false -->
		<!-- But adBlocker might stop it so it will still be true -->
		<script src="ads.js"></script>
		<script>
			// Try if adBlockerDectected is true
			if(adBlockerDectected) {
				/* 
				What switch the line of code below with Js code
				you want to activate when addblocker is dectected.
				*/
				console.log("AdBlocker detected!");
			}
		</script>
	</head>
	<body>
		<!-- Your page content here -->
	<body>
</html>

Solution Two

This solution is used if you are using Google Adsense. It checks (after the page has loaded using setTimeout), the existence of window,google_jobrunner.

  1. Step One
    Create a file called detectAdBlock.js, put the below content into it and link it to you html page,
    Or
    Put the below content directly into your html page in script tags.
  window.onload = function() {
    setTimeout(function() {
      if ( typeof(window.google_jobrunner) === "undefined" ) {
        console.log("AdBlocker detected!");
      } else {
        console.log("AdBlocker not detected.");
      }
    }, 10000);
  };

Solution Three

The solution uses jQuery and it has a bait, a div with the class adbanner. Most ad blockers will block it and if it does, then the Js code is dectecting if the height is zero which is what most adBlockers do.

  1. Step One
    Get to jQuery CDN (Or install it if you wish), by putting the below content into your head in html.
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  1. Step Two
    Add the main code, the finished code should look like this,
<!DOCTYPE HTML>
<html>
	<head>
		<!-- Scripts -->
		<!-- Add jQuery CDN -->
		<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
		<script>
			$(document).ready(function(){
				// Detect height
				if($("#wrapfabtest").height() > 0){
					console.log("AdBlocker detected!");
				} else {
					console.log("AdBlocker not detected.");
				}
			}
		</script>
		<!-- End scripts -->
		<!-- A bit of CSS but if you don't want to mess around with 	
		CSS then you can add non-breaking space to the element (the one inside the dic with wrapfabtest as it's class)
		Like this:
			<div class="adBanner">
				&nbsp;
		    </div>
		-->
		<style>
			.adBanner {
				background-color: transparent;
				height: 1px;
				widith: 1px;
			}
		<style>
	</head>
	<body>
		<!-- The fake ad -->
		<div id="wrapfabtest">
		    <div class="adBanner">
		    </div>
		</div>
		<!-- End fake ad -->
		<!-- Your other content -->
	</body>
</html>
